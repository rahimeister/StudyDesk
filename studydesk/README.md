#  StudyDesk - PyQt5 Productivity and Study Tracker

A modern desktop productivity application built with **Python**, **PyQt5**, and **SQLite** to help students organize study sessions, manage notes, track learning progress, and improve daily productivity.

![Python](https://img.shields.io/badge/Python-3-blue?logo=python)
![PyQt5](https://img.shields.io/badge/PyQt5-GUI-green)
![SQLite](https://img.shields.io/badge/SQLite-Database-blue)
![Status](https://img.shields.io/badge/Status-Completed-success)

---

##  Features

-  Dashboard with study overview
-  Pomodoro Timer (Start / Pause / Reset /finish and save)
-  Study Topic Management
-  Note Management
-  Productivity Statistics
-  SQLite Database Integration
-  Modern Desktop Interface

---

## 🛠 Technologies

- Python
- PyQt5
- SQLite
- Object-Oriented Programming (OOP)

##  Installation

Clone the repository

```bash
git clone https://github.com/rahimeister/StudyDesk.git
```

Move into the project directory

```bash
cd StudyDesk
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
