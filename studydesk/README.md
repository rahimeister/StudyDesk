# StudyDesk - PyQt5 Productivity and Study Tracker

StudyDesk; Pomodoro, ders/konu takibi, not alma, günlük hedef ve çalışma istatistiklerini bir araya getiren masaüstü verimlilik uygulamasıdır.

## Özellikler
- Modern koyu temalı PyQt5 arayüz
- SQLite veri kalıcılığı
- QTimer tabanlı Pomodoro
- Ders ve konu ekleme/tamamlama
- Günlük hedef ve ilerleme çubuğu
- Not ekleme ve TXT dışa aktarma
- Toplam süre, en çok çalışılan ders ve haftalık özet

## Kurulum
```bash
python -m venv .venv
```

Windows:
```bash
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

Linux/macOS:
```bash
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Teknolojiler
Python, PyQt5, SQLite, OOP, QTimer, QStackedWidget.
